Portfolio Website

For the Art of Web Programming course, I created this portfolio website from scratch. I used a little JQuery for the scrolling animations. As the years have gone on, I've really simplified the information architecture of the website, like combining the contact methods section with a short blurb about my interests, and combining highlights with my work experience. My favorite part was making the hover and active states for the project tiles, because I love built-in CSS features. The tiles are inspired by the iOS App Store graphic design. Thanks for checking out the website!
